# 📷 Camera Controller — Система настройки фона для сцен

## 📖 Описание

Этот скрипт позволяет настраивать **разный фон для каждой сцены** прямо в инспекторе Unity. Все настройки хранятся в компоненте `CameraController` на объекте камеры.

---

## 🚀 Как использовать

### 1️⃣ Базовая настройка

1. **Выберите объект камеры** в сцене (или создайте новый)
2. **Добавьте компонент** `Camera Controller` (если ещё не добавлен)
3. В инспекторе вы увидите следующие секции:

```
📷 Camera Controller
├── ⚙️ Основные настройки
│   └── 🎯 Цель (Transform)
├── 📐 Настройки позиции
│   ├── 🏠 Оффсет Hub (для MainScene)
│   └── 🌍 Оффсет по умолчанию
├── 🌄 Настройки фона сцен
│   ├── ✅ Авто-применение
│   ├── 📋 Сцены (список)
│   └── 🔁 Фон по умолчанию
├── 🎨 Быстрые пресеты
├── 🔧 Отладка
└── ⚡ Действия
```

---

## 🎨 Настройка фона для сцены

### Добавление сцены в список

1. В секции **🌄 Настройки фона сцен** нажмите **"➕ Добавить сцену"**
2. В появившемся элементе укажите:
   - **🏷️ Название сцены** — должно **точно совпадать** с именем сцены в Unity
   - **🎨 Тип фона** — выберите один из 4 типов:
     - **🎨 Цвет** — сплошной цвет фона
     - **🖼️ Текстура** — текстура на весь экран
     - **🖼️ Спрайт** — спрайт (изображение) на весь экран
     - **🌌 Скайбокс** — 3D скайбокс (Material)

### Настройка типов фона

#### 🎨 Цвет (Color)
- **🌈 Цвет фона** — выберите цвет через Color Picker
- Превью цвета отображается ниже

#### 🖼️ Текстура (Texture)
- **🖼️ Текстура** — назначьте Texture2D
- **📏 Масштаб** — размер текстуры (1-100)
- **🎨 Тинт** — цвет наложения (белый = без изменений)
- Превью текстуры отображается ниже

#### 🖼️ Спрайт (Sprite)
- **🖼️ Спрайт** — назначьте Sprite
- **🎨 Тинт** — цвет наложения
- Превью спрайта отображается ниже

#### 🌌 Скайбокс (Skybox)
- **🌌 Материал скайбокса** — назначьте Material с шейдером Skybox
- Превью материала отображается ниже

## ⚡ Быстрые пресеты

В секции **🎨 Быстрые пресеты** можно выбрать готовую настройку:

| Пресет | Описание | Цвет/Настройки |
|--------|----------|----------------|
| 🌑 Тёмная ночь | Тёмно-синий фон | `RGB(0.05, 0.05, 0.15)` |
| ☀️ Ясный день | Голубой фон | `RGB(0.53, 0.81, 0.92)` |
| 🌅 Закат | Оранжево-красный фон | `RGB(1.0, 0.4, 0.2)` |
| 🌫️ Туман | Серо-зелёный фон | `RGB(0.7, 0.75, 0.7)` |
| 🌌 Космос | Скайбокс | Требует ручной настройки материала |

## 🔧 Кнопки действий

| Кнопка | Описание |
|--------|----------|
| **🔄 Применить фон для текущей сцены** | Немедленно применяет настройки для активной сцены |
| **➕ Добавить настройку для текущей сцены** | Добавляет новую запись в список с именем текущей сцены |
| **🗑️ Очистить все настройки** | Удаляет все записи из списка (требует подтверждения) |

## ⚙️ Дополнительные настройки

### ✅ Авто-применение
Если включено — фон автоматически применяется при загрузке новой сцены.

### 🔁 Фон по умолчанию
Если сцена не найдена в списке, используется эта настройка.

### 🔧 Debug Mode
Включает отладочные сообщения в консоли Unity.

---

## 🎯 Советы

1. **Имена сцен чувствительны к регистру!** `Level1` ≠ `level1`
2. **Проверяйте имена сцен** в окне Build Settings (Ctrl+Shift+B)
3. **Используйте превью** для быстрой проверки настроек
4. **Сохраняйте префаб камеры** с настроенными сценами для повторного использования
5. **Для текстур** используйте режим Wrap Mode: Repeat для бесшовного фона

---

## 🐛 Решение проблем

| Проблема | Решение |
|----------|---------|
| Фон не применяется | Проверьте, что имя сцены точно совпадает |
| Текстура не видна | Убедитесь, что Texture Scale достаточно большой |
| Скайбокс не работает | Проверьте, что у материала правильный шейдер |
| Цвет не меняется | Убедитесь, что Camera.clearFlags установлен правильно |

---

## 📞 Поддержка

При возникновении проблем включите **Debug Mode** и проверьте консоль Unity на наличие сообщений, начинающихся с `[CameraController]`.
